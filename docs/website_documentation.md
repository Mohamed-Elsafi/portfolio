# Mohamed El-Safi Portfolio Website Documentation

## Overview
This documentation provides comprehensive information about your portfolio website, including its structure, features, and instructions for maintenance and updates.

## Website Structure
The website follows a modern, responsive design with the following structure:

### Pages
1. **Homepage (index.html)** - Main landing page with sections for:
   - Hero introduction
   - About Me
   - Featured Projects
   - Technical Skills
   - Professional Experience
   - Contact Information

2. **Project Pages** - Detailed case studies for key projects:
   - C-Ward Expansion Project (pages/c-ward-expansion.html)
   - Underground Water Tank Project (pages/water-tank.html)
   - Perimeter Fence Construction Project (pages/perimeter-fence.html)

### Directories
- **css/** - Contains all styling files
- **js/** - Contains JavaScript functionality
- **images/** - Contains all image assets
- **pages/** - Contains project detail pages
- **docs/** - Contains documentation

## Maintenance Guide

### Updating Content

#### Updating Text Content
1. Open the relevant HTML file in a text editor
2. Locate the section you want to update
3. Modify the text between the HTML tags
4. Save the file

#### Adding New Projects
1. Create a new HTML file in the `/pages` directory using an existing project page as a template
2. Update all content, images, and links
3. Add a link to the new project on the homepage in the "Projects" section

#### Updating Images
1. Prepare your new image (recommended size: 1200px width for large images)
2. Save the optimized image to the `/images` directory
3. Update the image path in the relevant HTML file

### Technical Maintenance

#### CSS Modifications
The main stylesheet is located at `/css/styles.css`. To modify styles:
1. Open the CSS file in a text editor
2. Locate the relevant section (sections are commented for easy navigation)
3. Modify the CSS properties as needed
4. Save the file

#### JavaScript Modifications
The main JavaScript file is located at `/js/main.js`. To modify functionality:
1. Open the JS file in a text editor
2. Locate the relevant function
3. Modify the code as needed
4. Test thoroughly before deploying
